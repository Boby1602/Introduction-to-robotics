# Simon Says
This project is a game like "Simon says". It has these components:
  1. 6 LEDs (3 blue, 1 green, 1 red, 1 yellow)
  2. 3 push-buttons
  3. 6 resistors
  4. 1 breadboard
  5. 1 arduino uno
